    SessionManager

    Api2 중 SessionManager는

    Event.NOTIFICATION_POSTED

    의 리스너에 전달되는 인자로, 채팅방 세션을 메신저봇 앱에 수동으로 등록할 수 있게 해줍니다. 즉, 메신저봇이 공식적으로 지원하지 않는 앱에 자동응답 봇을 탑재하고자 할 때 사용할 수 있습니다.

    action

    action

    이

    null

    값을 가질 경우, 메신저봇이 자동 분석한 답장 액션을 사용합니다.

    이

    null

    packageName

    을 자동 분석합니다.
